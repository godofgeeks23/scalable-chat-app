# Scalable Chat App\

## Setup

### Clone the repository

```
git clone https://github.com/godofgeeks23/scalable-chat-app.git
```

Install dependencies

```
cd server
npm install
cd ../client
npm install
```

## Run

### Start WebSocket (Socket.IO) server

```
cd server
node index.js
```

### Start the client

```
cd client
npm run dev
```
